# Project Intent

## 一言でいうと

自分のアバターモデルをできるだけ崩さず Android のライブ壁紙に置き、
背景や演出も含めてこちらから干渉できる
「俺の考えた最強の背景ソフト」を作りたい。

## このプロジェクトで本当にやりたいこと

- ただ VRM を壁紙に表示したいわけではない
- ただ 3D モデルを眺めたいわけでもない
- 自分のアバターモデルをホーム画面上に立たせたい
- タップなどの入力に対して、モデルが反応してほしい
- 背景も含めて、キャラが映える状態を自分で作りたい
- できるだけ元のモデルの見た目や雰囲気を壊したくない

## なぜ作るのか

- 既存の VRM 壁紙は「置くだけ」で終わるものが多い
- モデルにこちらから干渉できる余地が少ない
- VRChat 用のアバターを含めて、自分の好きなモデルをもっと自分向けに使いたい
- 壁紙を単なる背景ではなく、軽く触れる存在にしたい

## このプロジェクトの核

### 1. インタラクティブであること

- タップや入力に反応する
- 表情変更やリアクションが返る
- 可能なら部位ごとに反応を変えられる

### 2. モデルをできるだけ崩さないこと

- 取り込みの都合で見た目が大きく変わらない
- 元の雰囲気やシルエットを保ちたい
- 今は VRM を使っていても、目的は「VRM 対応」そのものではない

### 3. 背景も主役の一部であること

- のっぺりした背景で十分な場合もある
- 逆にキャラを立てるために背景演出が必要な場合もある
- 最終的にはモデルだけでなく背景込みで体験を作りたい

## 今の時点での優先順位

### 最優先

- モデルを読み込んで安定して表示できる
- Android ライブ壁紙として普通に動く
- タップなどの入力でモデルに反応を返せる
- 背景とカメラを最低限調整できる

### 次に重要

- モデルの取り込みフローをもっと楽にする
- 見た目の崩れを減らす
- 反応の種類を増やす
- 背景の表現を広げる

### 後回し

- 豪華な設定 UI
- 汎用 3D ビューア化
- 多人数向けの公開アプリ設計
- 何でもできる編集ツール化

## 非目標

- Unity の UI を頑張って作ること自体
- 最初から VRChat アバター完全互換を目指すこと
- 何でも読み込める万能ツールを最初に作ること
- 配布用に完璧なプロダクトにすること

## 今の仮の定義

このプロジェクトは、

「自分のアバターモデルを、見た目をなるべく保ったまま Android のライブ壁紙に置き、
入力に反応させ、背景込みで気持ちよく見せるための個人用ランタイム」

として考える。

## 今後の判断基準

新しい思いつきを入れる前に、次を確認する。

- それはモデルを気持ちよく立たせるために本当に必要か
- それはインタラクションを良くするか
- それは取り込みや見た目維持に効くか
- それとも単に作業だけ増やす脱線か

この基準に当てて、外れるものは後回しにする。
